# ajax 简介
### Asynchronous JavaScript and XML
- 中文翻译 -- 异步的 JavaScript 与 XML
- ajax 是前后端通信的桥梁
- 浏览器端不用刷新整个页面就可以与服务器端通信的技术
- 多技术的组合
	- JavaScript、HTML、CSS、DOM、XML、JSON 和 XMLHttpRequest
	
### ajax 的应用领域
- 搜索框（部分内容动态刷新）
- 地图
- 用户名检查
- 收货地址三级联动

### 区别传统的 WEB 请求与 AJAX 
#### 传统 WEB 请求
- 采用同步方式与后台交互
- 麻烦 -- 如果仅仅需要改变页面的某一部分的内容，也不得不刷新整个页面
