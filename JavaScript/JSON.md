# JSON 
### JavaScript Object Natation
- 中文翻译 -- JS 对象表示法
- JSON 是表示数据的一种格式
- JSON 实际就是一个字符串
- JSON 中的属性名和属性值都必须加双引号
- JSON 可以被任意的语言任意的平台所识别
- JSON 不能加注释

### 语法规范
- JSON 对象 -- '{}'
- JSON 数组 -- '[]'
- JSON 中可以使用的数据类型
	- String 字符串
	- Number 数值
	- Boolean 布尔值
	- null 空值
	- Object 对象
	- Array 数组

### JSON 工具类
- JSON.stringify()
	- JS 对象转换为 JSON 字符串
- JSON.parse()
	- JSON 字符串解析成 JS 对象
